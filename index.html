Catalog

What's out there?

Survey respondents noted that a part of the problem is knowing what exists. Long standing projects like snort, bro, suricata, nmap, metasploit benefit from tribal knowlege as well as coverage in certifcations, etc. New projects have things like demo lab at Defcon, arsenal at black hat, but get almost no coverage from press, etc unless they are attached to a tech giant like facebook, google, etc.

Pwnage?

Easily check data exposure: https://github.com/brycx/checkpwn

Using the k-anonymity mathmatical property to take the first x chars of the hash, get a list of 3-400 potential matches and search within that on the client side such that your password is never sent.

Binary Alert

Easy s3/yara https://github.com/airbnb/binaryalert

(relatively easy)

aws rube-goldberg machine made easy to send files to s3 buckets to be analyzed against your own set of yara signatures.

Falco

sysdig's falco

falco is syscall monitoring with really nice rulesets to gain visibility and alerting on actions within containers without having to modify the containers.
